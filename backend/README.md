# VeV_ecommerce
ecommerce to VeV 

# run project

* use to download project

~~~
git clone "repository"
~~~

* use to install dependences

~~~
yarn
~~~

* use to run project

~~~
yarn dev
~~~

# how to make new feature

* checkout master

~~~
git checkout master
~~~

* pull all updates

~~~
git pull origin master
~~~

* create a new branch

~~~
git checkout -b "branch_name"
~~~

* make modifications and push
* create e pull request by github
